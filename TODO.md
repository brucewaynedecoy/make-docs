# Starter Docs Tasks

[ ] TASK01: Come up with and implement a strategy for "building" supporting documents (like agent instructions, templates, and references) for document "parts". That way the parts can be maintained, and "building" the `docs` directory or `skills` directory will rebuild the supporting documents from the parts. This would also give the CLI/wizard the ability to build supporting documents from maintained parts (rather than hardcoding the document parts directly in files under `src/`).

[ ] TASK02: Compare templates and references in `skills/decompose-codebase` skill against `docs/.templates` and `docs/.references` respectively. Come up with a strategy for creating document parts for supporting docs unique to the `decompose-codebase` skill.  Implement into the supporting docs factory that should be the outcome of `TASK01`.

[ ] TASK03: Add a `docs/assets` directory (with agent instructions). This will contain 3 child directories (to start) at `docs/assets/examples`, `docs/assets/guides`, and `docs/assets/images` (each will also have its own set of agent instructions to follow the pattern of progressive disclosure).  The "examples" directory will be used for storing example code, example codebases, or anything else that an agent might derive useful patterns or antipatterns, or that an agent might use to get an idea of how to build, implement, or document something.  The "guides" directory will be used for storing user-facing reference guides and how-to documentation.  The "images" directory will be used for storing screenshots, images embedded in documentation, etc.

[ ] TASK04: Add instructions to root `AGENTS.md` and `CLAUDE.md` files on handling moved or removed `docs` directory (or child directories).  Instructions should instruct the agent on how to handle a situation when tracked directories aren't found (when the agent goes looking for them); essentially, the agent should go looking for them. If they aren't found (i.e., they've been deleted or moved into a directory that the agent can't read), the agent should ask the user if they would like to recreate the missing directories. If they are found but have been moved to a different location, the agent should ask the user for permission to create agent instruction files (e.g., `AGENTS.md` and/or `CLAUDE.md`) to guide the agent to the directories.
