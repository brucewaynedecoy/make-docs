# Legacy History Records

This directory contains legacy point-in-time session breadcrumbs. Do not create new history records here.

New history records belong under `docs/.assets/history/`.

## Agent Instructions

- For new history records, read `docs/.references/history-record-contract.md`, `docs/.templates/history-record.md`, and the router in `docs/.assets/history/`.
- Write a new record in this legacy directory only when the user explicitly asks to avoid dogfooding `docs/.assets/history/`.
- Edit legacy records only to correct factual errors or broken links.
- Do not append new session work to a prior session's summary.
