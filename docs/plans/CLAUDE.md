# Plans Router

This directory is an output target for reviewable plans created before execution.
- Files use `YYYY-MM-DD-<slug>.md`; use today's date and never backdate.
- Before writing, read `docs/.references/planning-workflow.md` and choose the matching plan template in `docs/.templates/`.
- Use `docs/.references/output-contract.md` for output paths and `docs/.references/prd-change-management.md` when the plan evolves an active PRD namespace.
- Treat the selected plan template as the authority for plan structure.
