name: Designs to Plan Change
description: Instructs the agent to classify a requested additive change, enhancement, revision, or removal and create a detailed change plan for the active PRD namespace.
___

Please review the design doc {{DESIGN DOC}} which represents a change request. Then inspect the active PRD namespace in `docs/prd/` and create a detailed change plan in `docs/plans/`. Follow the instructions, references, and templates in the `docs` directory, especially `docs/.references/prd-change-management.md`. The plan should classify the request as an addition, enhancement, revision, or removal; identify the impacted baseline docs; define the new PRD change docs that should be created; define the required baseline annotations; and state whether a scoped delta backlog is sufficient.
